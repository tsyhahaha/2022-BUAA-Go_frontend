# go_frontend

本仓库是 2022 年 BUAA 数据库大作业体育课程网站《走马Go》的前端部分，本实现基于 vue。vue 基本配置请参考[官网](https://cli.vuejs.org/config/).

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
